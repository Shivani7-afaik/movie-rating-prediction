# Movie Rating Prediction

This project predicts movie ratings based on features such as genre,
duration, and number of votes using regression techniques.

## Dataset
IMDb Movies India dataset containing information about movies including
genre, duration, votes, and ratings.

## Approach
- Data cleaning and preprocessing
- Feature engineering
- Encoding categorical variables
- Linear Regression modeling
- Model evaluation using MAE, RMSE, and RÂ² score

## Results
The model predicts movie ratings with reasonable accuracy and provides
insight into factors influencing ratings.
